# Claude Monet Collection

## Scope

This collection contains the known paintings by Claude Monet.

## Copyright Status

**Public Domain Worldwide.** 
Claude Monet died on December 5, 1926. This is significantly more than 70 years ago, meaning his entire body of work is in the public domain globally, adhering to the project's copyright rules.

## Metadata specifics
- **Era**: Period in Monet's life (Early years, Argenteuil, Vétheuil, Giverny)
- **Series**: Monet painted extensively in series (Water Lilies, Haystacks, Rouen Cathedral, Poplars, etc.). This is captured in the dataset for UI filtering.
- **W Number**: The standard catalog raisonné index by Daniel Wildenstein.
